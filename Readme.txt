Github URL for clone de project: 
	https://github.com/joarguma/DevsuE2E.git

1. After clone the project
2. Go to path that you clone the solution
3. Open folder [DevsuE2E]
4. Open File [DevsuE2E.sln]
5. When the solution is Open, press ctrl + R + A after executing (On your left you are going to see the execution result about the indicate test in the requirement)
	Another way to execute the test, go to the option test and them click on "Run All Tests"

Nota:
	Al momento de la ejecucion de los casos, el endpoint de registro falla, volver a ejecutar nueva vez pues se coloco un incrementador para que los usuarios no se repitan sin la necesidad de manipular las variables.
